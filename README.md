# LangChain Demo
Exploring LangChain capabilities


## Environment
The project use __uv__ as a package manager. Use __.venv__ as project virtual enviroment.

```bash
# Activation
source .venv/bin/activate

# Deactivation
deactivate
```

### Package management

```bash
# cli app
uv add <package>
uv remove <package>

# python package
uv pip install <package>
uv pip uninstall <package>
```
